개인 과제 할일 목록

## 필수 기능

1.  [] 캐릭터 이동 및 맵 탐색
  - [] WASD 혹은 방향키를 사용해 2D 캐릭터 이동.
  - [] Rigidbody2D와 Collider2D를 이용.
  - [] 캐릭터가 맵의 가장자리에서 정지.

2.  [v] 맵 설계 및 상호작용
  - [v] 타일맵 혹은 간단한 오브젝트를 통한 맵 구성.
  - [] 상호작용이 가능한 오브젝트(NPC, 미니 게임 존 등) 배치.
  - [] 특정 영역 진입 시, 이벤트 트리거 작동.

3.  [] 미니 게임 실행
  - [] 특정 영역에서 미니 게임 실행.
  - [] 장애물 회피 미니게임(Flappy Plane) 혹은 블록 쌓기 미니게임(The Stack) 구현.
  - [] 미니 게임 시작 전, 게임 설명 UI 실행.
  - [] 미니 게임 종료 후 점수 반환 및 맵 복귀.

4.  [] 점수 시스템
  - [] 미니 게임 점수 기록 및 UI를 통한 실시간 표시.
  - [] 점수 기록 & 최고 점수 저장 및 표시.
  - [] 미니 게임 종료 후 맵에 돌아왔을 때 점수 유지되도록 저장.

5.  [] 게임 종료 및 복귀
  - [] 미니 게임 종료 시, 맵 복귀 및 결과 표시.
  - [] 미니 게임 성공/실패 메시지 표시.
  - [] 점수 등 게임 결과 UI 출력.

6.  [] 카메라 추적 기능
  - [] 캐릭터 이동에 따른 카메라 추적.
  - [] Transform 기능 사용.
  - [] 카메라가 특정 역역을 벗어나지 않도록 경계 설정.


## 도전 기능

7.  [] 미니 게임 추가
  - [] 새로운 미니 게임 추가.
  - [] 게임 규칙과 플레이 방식 설계.
  - [] 각 미니 게임의 고유 점수 계산 방식 추가.

8.  [] 캐릭터 커스텀
  - [] 캐릭터의 외형(색상, 장식 등) 변경 기능 추가.
  - [] 캐릭터 색상 변경 메뉴 제공.
  - [] 캐릭터 외형 변경 메뉴 제공.
  - [] 아이템 수집을 통한 장식 아이템 추가.

9.  [] 리더보드 시스템
  - [] 최고 점수 기록 및 UI 표시.
  - [] 최고 점수 PlayerPrefs 저장.
  - [] 로컬 리더보드 구현.

10. [] 탑승물 제작
  - [] 캐릭터의 이동 및 미니 게임에 활용할 수 있는 탑승물 추가.
  - [] 탑승 중 속도 증가, 장애물 통과 등 새로운 기능 제공.
  - [] 탑승물 고유 능력(속도, 방어력, 공격력 등) 추가.
  - [] 탑승물 업그레이드 및 구매 기능 (상점) 추가.

11. [] NPC 대화 시스템
  - [] NPC와 상호작용 시 대화창 활성화.
  - [] NPC와 충돌 시 대화창 표시.
  - [] 대화에 따라 힌트 제공 또는 이벤트 발생.
  - [] 플레이어 선택지 제공 및 선택에 따른 대화 내용 변화.
